# Weather App

A weather app built with React and Vite that fetches real-time weather data using the OpenWeatherMap API.

## Features
- Search weather by city name
- Displays temperature, humidity, wind speed, and feels like
- Press Enter or click Search to fetch data
- Error handling for invalid cities

## Tech Stack
- React
- Vite
- OpenWeatherMap API

## What I Learned
- Fetching data from a REST API using `fetch()`
- Handling asynchronous code with `async/await`
- Error handling with `try/catch`
- Managing multiple states with `useState`

## How to Run
1. Clone the repo
2. Run `npm install`
3. Add your OpenWeatherMap API key in `App.jsx`
4. Run `npm run dev`
